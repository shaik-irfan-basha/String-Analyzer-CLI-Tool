# String Analyzer CLI Tool
### Part of JASIQ Labs Internship (Work + Learning, Job Ready Internship)

A simple command-line interface tool to analyze strings, providing character count, word count, vowel count, reversal, and palindrome checks.

## Features:
- **Total characters**: Count of all characters in the string.
- **Total words**: Use `.split()` to count words.
- **Vowels**: Count of vowels (a, e, i, o, u) in the string.
- **Reversed**: The string in reverse order.
- **Palindrome**: Checks if the string reads the same forwards and backwards (case-insensitive).

## Usage:
Run the script using Python:
```bash
python string_analyzer.py
```
Input a string and get the analysis results.

## Sample:
Input: "madam"
Total characters: 5
Total words: 1
Vowels: 2
Reversed: madam
Palindrome: Yes
